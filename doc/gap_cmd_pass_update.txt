gap_cmd_pass_update(3)
=====================


NAME
----
gap_cmd_pass_update - sends passkey update command


SYNOPSIS
--------
*int gap_cmd_pass_update (cc2540_t *'dev', uint16_t 'handle', const char 'pass'[GAP_PASS_STR_LEN]);*


DESCRIPTION
-----------
*gap_cmd_pass_update ()* function sends the passkey update command to
the _dev_ device to update the passkey by using the _pass_ string.

The _pass_ should be ASCII string of numbers.


RETURN VALUE
------------
*gap_cmd_pass_update ()* returns 0 for success, and an _errno_ like
negative number for failure (_errno_ is also set appropriately).
