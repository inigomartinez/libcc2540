gap_cmd_dev_addr_set(3)
======================


NAME
----
gap_cmd_dev_addr_set - sends the configure device address command


SYNOPSIS
--------
*int gap_cmd_dev_addr_set (cc2540_t *'dev', gap_addr_t 'addr_type', const uint8_t 'addr'[BT_ADDR_LEN]);*


DESCRIPTION
-----------
*gap_cmd_dev_addr_set ()* function sends the configure device address command
to the _dev_ device setting its address to _addr_ being a _type_ address type.

The list of available address types is as follows:

GAP_ADDR_PUBLIC
~~~~~~~~~~~~~~~
Public address type.

GAP_ADDR_STATIC
~~~~~~~~~~~~~~~
Static address type.

GAP_ADDR_PRIV_NON_RESOLV
~~~~~~~~~~~~~~~~~~~~~~~~
Non resolvable private address type.

GAP_ADDR_PRIV_RESOLV
~~~~~~~~~~~~~~~~~~~~
Resolvable private address type.


RETURN VALUE
------------
*gap_cmd_dev_addr_set ()* returns 0 for success, and an _errno_ like
negative number for failure (_errno_ is also set appropriately).
