gap_evt_dev_init_done(3)
=======================


NAME
----
gap_evt_dev_init_done - read a device init done event


SYNOPSIS
--------
*int gap_evt_dev_init_done (cc2540_t *'dev', gap_evt_dev_init_done_t *'evt');*


DESCRIPTION
-----------
The *gap_evt_dev_init_done ()* function shall fill the 'evt' event from the
_dev_ device.


RETURN VALUE
------------
*gap_evt_dev_init_done ()* returns 0 for success, and an _errno_ like negative number for
failure (_errno_ is also set appropriately).


ERRORS
------
*ENOMSG* If the data read does not correspond to a gap_evt_dev_init_done event.
